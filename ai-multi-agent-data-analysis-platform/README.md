# AI Multi-Agent Data Analysis Platform

## Project Overview
This project is an AI-powered data analysis platform that automates the process of analyzing datasets using multiple intelligent agents.

Users can upload a dataset and the system will automatically:
- Clean the data
- Analyze the dataset
- Generate visualizations
- Train machine learning models
- Answer questions about the dataset

The goal of this project is to simulate how modern AI-powered analytics tools work.

---

## Features

### 1️⃣ Data Cleaning Agent
- Removes duplicate rows
- Handles missing values

### 2️⃣ Data Analysis Agent
- Shows dataset statistics
- Displays number of rows and columns
- Calculates average values

### 3️⃣ Visualization Agent
- Automatically generates graphs
- Helps understand data distribution

### 4️⃣ AutoML Agent
Automatically trains machine learning models and selects the best model.

Models used:
- Linear Regression
- Random Forest

### 5️⃣ AI Chat Agent
Allows users to ask questions about the dataset.

Example questions:
- How many rows are in the dataset?
- Show summary
- What is the average purchase amount?
